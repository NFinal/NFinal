QueryTop方法

QueryTop(string sql，int top) 有两个参数 sql:为sql查询语句 top:需要获得的行数
返回：行对象数组

1、无参数 
    public void  Index()
    {
        //打开一个数据库链接
        var con= Models.YMC.OpenConnection();
        var rows= Models.YMC.QueryTop("SELECT * FROM user",3);
        //关闭数据库连接
        con.Close();
    }

模板会根据以上代码生成正确的执行代码，此处不再显示。可以参见QueryRandom中的模板生成的代码

2、有默认值的参数 
    public void  Index(string pwd="123456")
    {
        //打开一个数据库链接
        var con = Models.YMC.OpenConnection();
        var row = con.QueryTop("SELECT * FROM user WHERE pwd=@pwd"，3);
        //关闭数据库连接
        con.Close();
    }
