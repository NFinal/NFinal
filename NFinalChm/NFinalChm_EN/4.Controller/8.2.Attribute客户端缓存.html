Attribute客户端缓存

使用缓存之前，必须先引入NFinal.Advanced;命名空间

根据http头Cache-Control详解中的三种情况，浏览器端缓存分为四种情况。

Browser对应的枚举值 说明
NoStore 表示永不缓存，即第一种情况
Expires 一定时间内访问缓存,即第二种情况
NotModify 文件未修改就一直使用缓存，即第三种情况
NoExpires 表示永远缓存，一般jquery,css样式及静态文件采用此选项 

        //urlPrefix="/App";
        namespace WebSite.App.Controllers
        {
            public class IndexController
            {
				//Server.NoCache表示服务端不缓存
				//Browser.NoStore表示浏览器缓存，缓存时间为100分钟。
				[Cache(Server.NoCache,Browser.Expires,6000)]
                public void Index()
                {
                    Write("Hello!");
                }
            }
        }
