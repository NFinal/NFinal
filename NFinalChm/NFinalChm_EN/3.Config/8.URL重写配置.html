URL重写

config.json中的rewriteDirectory,rewriteFile

目录重写,把/1开始的url全改为以/news（文章）开头 
    "rewriteDirectory": [
        {"from":"/news/","to":"/1/"}
    ]
文件重写,把/1/1.html重写为/1/index.html 
    "rewriteFile": [
		{ "from":"/1/1.html", "to":"/1/index.html" }
	  ]

经过上面两个规则，请求的路径
/news/index.html被重写为/1/1.html
/news/2.html被重写为/1/2.html
/news/3.html被重写为/1/3.html
....
/news/n.html被重写为/1/n.html
