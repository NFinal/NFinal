Session配置

1.在[模块名]/Code/Data/SessionInfo.cs中定义你的Session变量 
        public class SessionInfo
        {
            public id userId=0;
        }
2.运行WebCompiler.aspx生成[模块名]/Code/Data/SessionManager.cs类
3.在Controllers中读取和使用Session: 
        public void Index()
        {
            //让Session操作变得更简单.
            Session.userId=2;
            Write(Session.userId);
        }

此方式的好处有3：
1.便于Session统一管理。
2.方便使用，不用去特意记忆Session名称，只需按.就会有自动提示。
3.当调用Web层下的Action类生成静态页时，只需把Session作为参数填进去即可。
