Cookie配置

1.在[模块名]/Code/Data/CookieInfo.cs中定义你的Cookie变量 
        public class CookieInfo
        {
            public string name="";
            public bool islogin=false;
            public int age=0;
        }
2.运行WebCompiler.aspx生成[模块名]/Code/Data/CookieManager.cs类
3.在Controllers中读取和使用cookie: 
        public void Index()
        {
            //让Cookie操作变得更简单.
            Cookie.name="NFinal";
            Write(_cookies.age);
        }

此方式的好处有3：
1.便于Cookie统一管理。
2.方便使用，不用去特意记忆Cookie名称，只需按.就会有自动提示。
3.当调用Web层下的Action类生成静态页时，只需把cookie作为参数填进去即可。
