QueryRow方法

QueryAll(string sql) 有一个参数 sql:为sql查询语句
返回：行对象

1、无参数： 
    public void  Index()
    {
        //打开一个数据库链接
        var con = Models.YMC.OpenConnection();
        var row = con.QueryRow("select * from user where id=23");
        //关闭数据库连接
        con.Close();
    }

模板会根据以上代码生成正确的执行代码，此处不再显示。可以参见QueryRandom中的模板生成的代码

2、有默认值的参数： 
         public void Index(int id=23)
        {
            //打开一个数据库链接
            var con = Models.YMC.OpenConnection();
            var row = con.QueryRow("select * from user where id=@id");
            //关闭数据库连接
            con.Close();
        }
3、vs2015支持的新语法，让SQL语句可以有自动提示 
        Models.Tips.Common.users users = new Models.Tips.Common.users();
        public void queryRow(int id)
        {
            var con = Models.Common.OpenConnection();
            var user = con.QueryRow($"select {users.id},{users.name},{users.pwd} from {users} where {users.id}=@id");
            con.Close();
            AjaxReturn(user);
        }

模板会根据以上代码生成正确的执行代码，此处不再显示。可以参见QueryRandom中的模板生成的代码
