Url生成

Url采用函数生成（URL规则一旦改变，代码可保持不变）

后端生成
		public void Index()
		{
			//生成URL
			string url=Url.App_IndexController_Index();
		}
        //对应的Url类位于根目录下的Url.cs文件中，
		//此文件由生成器自动生成。
		//模块名：App
		//类名为:IndexController
		//方法名:Index
前端生成 
        //需要引入网站根目录下的Url.js文件
        var url=Url.App_IndexController_Index();
		//对应的Url类位于根目录下的Url.js文件中，
		//此文件由生成器自动生成。
		//模块名：App
		//类名为:IndexController
		//方法名:Index

如果Url中有参数，则Url对应的生成函数也会有相应的参数。
例：
string url=Url.App_IndexController_News(int page);
