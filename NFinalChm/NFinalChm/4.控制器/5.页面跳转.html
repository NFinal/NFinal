页面跳转

Extesion.cs类提供了两个函数,Success和Error 我们可以在控制器中直接调用.如下: 

        if(isLogin)
        {
			//参数1：显示信息，参数2为跳转后的URL，参数3为跳转所需的秒数
            this.Success("登陆成功", "/App/AdminController/Index.htm", 3);
        }
        else
        {
			//参数1：显示信息，参数2为跳转后的URL，参数3为跳转所需的秒数
            this.Error("登陆失败","/App/UserController/Login.htm",3);
        }

对应的模板位于Views/Default/Common/Public/下 分别是Success.aspx和Error.aspx,用户可以重新修改其样式.
