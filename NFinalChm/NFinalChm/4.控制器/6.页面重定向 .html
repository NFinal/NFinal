页面重定向

与Success()和Error()两个函数的使用类似 我们可以在控制器中直接调用.如下: 

        if(isLogin)
        {
            this.Redirect("/App/AdminController/Index.htm");
        }
        else
        {
            this.Redirect("/App/UserController/Login.htm");
        }

对应的模板位于Views/Default/Common/Public/下 分别是Index.aspx和Login.aspx,用户可以重新修改其样式.
