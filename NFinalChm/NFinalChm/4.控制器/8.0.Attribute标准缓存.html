Attribute缓存

使用缓存时，请先引入NFinal.Advanced命名空间
缓存函数为：

缓存特性 说明
[Cache(Server server,Browser browser,int second)] 使用缓存，
参数1为服务器缓存设置，
(NoCache,FileDependency,AbsoluteExpiration,SlidingExpiration)
参数2为浏览器缓存设置，
(NoStore,NotModify,Expires,NoExpires)
参数3为缓存的时间
[Cache(Standard standard,int second)] 使用标准缓存，
参数1为一般缓存设置
(其中有File和Nomal两种，一种是文件缓存，一种是内存缓存)，
参数2为缓存的时间
[CacheFile(int second)] 使用文件缓存，参数为缓存的时间
[CacheNormal(int second)] 使用内存缓存，参数为缓存的时间

文件缓存 
        //urlPrefix="/App";
        namespace WebSite.App.Controllers
        {
            public class NewsController
            {
				[CacheFile(6000)]
                public void Index()
                {
                    Write("Hello!");
                }
            }
        }
        //缓存100分钟，缓存文件将会存储于模块文件夹下的HTML中。
		//当该文件改变时缓存将会失效。
内存缓存 
        //urlPrefix="/App";
        namespace WebSite.App.Controllers
        {
            public class NewsController
            {
				[CacheNormal(6000)]
                public void Index()
                {
                    Write("Hello!");
                }
            }
        }
        //缓存100分钟，缓存文件将会存储于内存中。
		//当重启IIS时缓存将会失效。
