基本视图语法

        //1、@{}中可直接写csharp代码.
        @{ int a=0; }
        //2、@name 输出变量
        @a
        //3、ViewBag.变量名(访问控制器中的变量)
        @ViewBag.title

在视图中可以直接输出控制器方法内的变量以及方法中的参数

    public class IndexController:Controller
    {
        public void Index(string name)
        {
            string word="hello ";
            View("Index.aspx");
        }
    }

运行生成器Web.Compiler.aspx,则可直接生成Razor模板Index.cshtml
以及Index.cshtml.cs模型文件，当然，你也可以自己手动创建。但必须要声明一个指向model的ViewBag变量。 

        @{var ViewBag=new WebApp.App.Views.Default.IndexController.Index_AutoComplete();}
        <!DOCTYPE html>
        <html>
        <head>
                <meta name="viewport" content="width=device-width" />
                <title></title>
        </head>
        <body>
            @ViewBag.word @ViewBag.name
        </body>
        </html>

Index.cshtml.cs如下 

        using System;
        using System.Collections.Generic;
        namespace WebApp.App.Views.Default.IndexController
        {
	        //数据库模型
	        //函数内变量
            public class Index_AutoComplete:Controller
            {
		        /*参数变量*/
                public string name;
		        /*数据库变量*/
		        /*一般变量*/
                public string word;
		        /*DAL函数声明变量*/
            }
        }
